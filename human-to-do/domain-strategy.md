# Domain Strategy: windowtintlaws.us + tintlaws.com

**Time needed**: 1-2 hours for research, then decision
**Priority**: MEDIUM — Important for long-term authority building

---

## Step 1: Check Domain Rating (DR) of Both Domains

Use one or more of these free tools:

### Option A: Ahrefs Free Backlink Checker
1. Go to [https://ahrefs.com/backlink-checker](https://ahrefs.com/backlink-checker)
2. Enter `windowtintlaws.us` → note the **Domain Rating (DR)**, **Backlinks count**, **Referring domains**
3. Enter `tintlaws.com` → note the same metrics
4. Enter `myeyerx.net` → note for comparison

### Option B: Moz Link Explorer
1. Go to [https://moz.com/link-explorer](https://moz.com/link-explorer)
2. Enter each domain
3. Note the **Domain Authority (DA)**, **Linking Domains**, **Inbound Links**

### Option C: Semrush (if you have access)
1. Enter each domain in Semrush Domain Overview
2. Note **Authority Score**, **Backlinks**, **Referring Domains**

## Step 2: Check What Links Point to tintlaws.com

Since you said tintlaws.com "had a lot of links pointing to it":

1. In Ahrefs Free Backlink Checker, enter `tintlaws.com`
2. Look at the **top referring domains** — are they quality sites or spam?
3. Check if any of these links are from .gov, .edu, or high-authority automotive/legal sites
4. Note the **anchor text** — is it relevant to window tint topics?

---

## Step 3: Decision Matrix

### Scenario A: tintlaws.com has significantly higher DR than windowtintlaws.us
**Recommendation**: Make tintlaws.com the primary law domain
- 301 redirect ALL pages on windowtintlaws.us → matching pages on tintlaws.com
- Build out tintlaws.com with state law content
- Cross-link from tintlaws.com ↔ myeyerx.net

### Scenario B: windowtintlaws.us has higher DR or they're similar
**Recommendation**: Keep windowtintlaws.us as primary (it's already active)
- 301 redirect tintlaws.com → windowtintlaws.us (entire domain)
- This transfers the backlink value from tintlaws.com to windowtintlaws.us
- Cross-link from windowtintlaws.us ↔ myeyerx.net

### Scenario C: Both have low DR (under 10)
**Recommendation**: Pick the better domain name and consolidate
- tintlaws.com is shorter and a .com (generally better)
- 301 redirect the other domain entirely
- Build fresh content on the chosen domain

---

## Step 4: Set Up 301 Redirects

Once you decide which domain to keep:

### If redirecting tintlaws.com → windowtintlaws.us:
1. Go to your domain registrar for tintlaws.com
2. Set up domain forwarding / 301 redirect:
   - Forward `tintlaws.com` → `https://windowtintlaws.us`
   - Forward `www.tintlaws.com` → `https://windowtintlaws.us`
   - Make sure it's a **301 (permanent)** redirect, not 302

### If redirecting windowtintlaws.us → tintlaws.com:
- Same process but reversed

---

## Step 5: Cross-Linking Strategy

Once consolidation is done, myeyerx.net law pages already link to the surviving domain. The surviving domain should link BACK to myeyerx.net on every page:

- Footer link: "Need a medical exemption? Get one at MyEyeRx.net"
- On each state law page: "Get your [State] medical tint exemption → myeyerx.net/[state-slug]"
- This creates a two-way authority loop between your own properties

---

## Do NOT Do These Things
- Do NOT redirect either domain to myeyerx.net (keep them separate — different topics)
- Do NOT build both law domains independently (splits authority)
- Do NOT use both domains for the same content (duplicate content penalty)
- Do NOT let tintlaws.com sit parked — either redirect it or build on it
